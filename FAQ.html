คำถามที่พบบ่อยและช่องทางติดต่อ

การเพาะต้นกล้า ถือเป็นขั้นตอนพื้นฐานและสำคัญยิ่งในกระบวนการปลูกพืชทุกประเภท เนื่องจากเป็นระยะเริ่มต้นของการเจริญเติบโต ซึ่งมีอิทธิพลโดยตรงต่อความแข็งแรง คุณภาพ และผลผลิตโดยรวมของพืชในระยะต่อไป หากต้นกล้ามีลักษณะสมบูรณ์และแข็งแรง จะช่วยให้พืชสามารถปรับตัวเข้ากับสภาพแวดล้อมใหม่ได้ดีเมื่อนำไปปลูกลงแปลงจริง ส่งผลให้การเติบโตเป็นไปอย่างต่อเนื่องและมีประสิทธิภาพสูง

ในทางตรงกันข้าม หากต้นกล้าอ่อนแอ เติบโตไม่สม่ำเสมอ หรือมีปัญหาทางสรีรวิทยา เช่น การเหี่ยวเฉา จะทำให้ผลผลิตลดลง สูญเสียเวลา และเพิ่มต้นทุนการผลิตโดยไม่จำเป็น จากการสังเกตและทดลองเพาะต้นกล้าแบบทั่วไป พบว่าต้นกล้าแต่ละต้นมักเจริญเติบโตไม่เท่ากัน บางต้นสูงเร็ว บางต้นไม่เติบโต หรือบางต้นเหี่ยวและตายก่อนย้ายปลูก

สาเหตุหลักมาจากความไม่สม่ำเสมอของปัจจัยแวดล้อม เช่น ความชื้นในดิน อุณหภูมิ แสงสว่าง และปริมาณน้ำ ซึ่งเป็นองค์ประกอบสำคัญต่อการเจริญเติบโต โดยเฉพาะในระยะต้นกล้าที่มีความเปราะบาง วิธีการเพาะแบบดั้งเดิมที่พึ่งพาแรงงานมนุษย์ เช่น การรดน้ำด้วยมือ หรือการวางถาดเพาะกลางแจ้ง ยังมีข้อจำกัดในการควบคุมปัจจัยเหล่านี้ให้คงที่

ด้วยเหตุนี้ คณะผู้จัดทำจึงได้ออกแบบและสร้าง “เครื่องเพาะปลูกต้นกล้ากระถางแบบครบวงจร” ซึ่งเป็นระบบควบคุมอัตโนมัติที่สามารถจัดการน้ำ ความชื้น อุณหภูมิ และแสงสว่างได้อย่างเหมาะสม ช่วยให้ต้นกล้าได้รับสภาพแวดล้อมที่สม่ำเสมอ ลดภาระการดูแล ลดการสูญเสียน้ำ และเพิ่มอัตราการรอดชีวิตของต้นกล้า ส่งเสริมให้การเพาะปลูกมีประสิทธิภาพมากยิ่งขึ้นในระยะยาว

ขั้นตอนการตั้งค่าเครื่องเพาะปลูกต้นกล้า

1. เสียบเครื่องเพาะปลูกต้นกล้าเข้ากับแหล่งจ่ายไฟ
2. เปิดไฟล์โค้ด Arduino ของ ESP8266
3. แก้ไขค่า WiFi SSID (username) และ Password ให้ตรงกับเครือข่ายของผู้ใช้
4. กรอก user_id ให้ตรงกับบัญชีของผู้ใช้ในระบบ
5. อัปโหลดโค้ดลงบอร์ด ESP8266
6. รีสตาร์ทบอร์ดและเริ่มใช้งานได้ทันที

1. เปิดไฟล์โค้ด Arduino ของ ESP32-Webcam
2. แก้ไขค่า WiFi SSID และ Password ให้ตรงกับเครือข่ายของผู้ใช้
3. อัปโหลดโค้ดลงบอร์ด ESP32-Webcam
4. รีสตาร์ทบอร์ดและเริ่มใช้งานได้ทันที

ขั้นตอนการเข้าสู่ระบบ
ผู้ใช้งานสามารถเข้าใช้งานระบบได้ที่ http://bit.ly/4skTm91
จากนั้นทำการ Log in เพื่อเข้าสู่ระบบ หากยังไม่มีบัญชีให้กดสมัครสมาชิก เมื่อลงทะเบียนเสร็จแล้วให้กลับมาที่หน้า Log in เพื่อเข้าสู่ระบบอีกครั้ง

เมื่อเข้าสู่ระบบแล้ว หน้าแรกของเว็บไซต์จะแสดงปุ่ม “เข้าสู่เว็บไซต์” เมื่อกดแล้ว ระบบจะแสดงหน้าควบคุม แขนกลที่ 1 (แขนกลสีเทาฝั่งขวา)

การใช้งานหน้าแขนกล 1
- ตารางเลือกดูต้นกล้า: กดเลือกจุดที่ต้องการดูข้อมูล
- ระบบจะแสดงตำแหน่งต้นกล้า ชื่อต้นกล้าที่ผู้ใช้ตั้ง รูปภาพที่อัปโหลด และกราฟค่าความชื้นที่วัดได้
- มีปุ่มควบคุม ได้แก่ คืนต้นกล้า / รดน้ำต้นกล้า / วัดความชื้น
- การตั้งค่าการทำงานสามารถดูได้ที่เมนู “การตั้งค่าการทำงานของเครื่องเพาะปลูกต้นกล้า”

เมื่อกดไอคอน 3 จุดที่มุมซ้ายของจอ จะเข้าสู่ระบบควบคุมอื่น ๆ ดังนี้

1. แขนกล 2
มีรูปแบบการใช้งานคล้ายกับแขนกลที่ 1 สามารถดูข้อมูลต้นกล้าแต่ละจุด และควบคุมการคืนต้นกล้า รดน้ำ และวัดความชื้นได้เช่นเดียวกัน

2. ระบบอัตโนมัติ
สำหรับตั้งค่าการทำงานอัตโนมัติ เช่น การรดน้ำ การเปิด-ปิดแผงโซลาร์เซลล์ หรือการเปิด-ปิดไฟ ผู้ใช้สามารถตั้งเวลา และกด Save เพื่อบันทึกการตั้งค่า

3. รูปภาพ
แสดงภาพจากกล้องที่ติดตั้งภายในเครื่อง เพื่อบันทึกการเจริญเติบโตของต้นกล้า สามารถกดปุ่มโหลดใหม่เพื่อดูภาพล่าสุดได้

4. ฟังก์ชันอื่น ๆ
สำหรับแอดไลน์เพื่อรับข้อมูลแบบเรียลไทม์ ดูคำอธิบายการใช้งานแบบย่อ และส่งความคิดเห็นถึงผู้พัฒนา
